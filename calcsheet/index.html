CalcSheet Formulas & QUERY Help

Basic Operators

Begin equations with =. Support cell coordinates (e.g. A1, B3, C10) and standard math signs: +, -, *, /, and parentheses ( ).

Example: =A1 + B2 * (C3 / 2)

Functions

Perform calculations over cell ranges (specified as StartCell:EndCell):

  • =SUM(range): Sums all numerical values in range.
  • =AVERAGE(range): Finds average of all numerical values.
  • =MIN(range): Finds the minimum value.
  • =MAX(range): Finds the maximum value.

Example: =SUM(A1:B10)

SQL-like QUERY(range, "query")

Executes a query on data tables. Columns in the range are mapped to Col1, Col2, Col3, etc.

Supported Clauses:

  • SELECT ColA, ColB (or SELECT *)
  • WHERE ColA > 10 AND ColB === 'Active' (standard JS logic)
  • ORDER BY ColA [ASC | DESC]

Example: =QUERY(A1:C10, "SELECT Col1, Col3 WHERE Col2 > 25 ORDER BY Col3 DESC")

Note: Results will automatically "spill" vertically and horizontally into adjacent cells starting from the formula cell.
