# UtilityHub: New Apps & Game Ideas

This document outlines 20 utility app ideas and 10 puzzly/shooter game ideas designed for the UtilityHub workspace.

---

## Part 1: 20 Utility App Ideas

### 1. RegExp Lab (Visual Regex Tester)
* **Description**: An interactive regular expression builder and tester.
* **Key Features**: Real-time match highlighting in test text, graphical syntax tree breakdown showing how the regex matches, group capture breakdown table, and a quick-reference cheat sheet for common patterns.

### 2. SVG Optimizer & Editor
* **Description**: A vector graphics minifier and visual tweaker.
* **Key Features**: Real-time drag-and-drop comparison of original vs. optimized SVGs, optimization toggles (remove comments, minify paths, clean IDs), basic path color swapping, and download support.

### 3. Palette Crafter (A11y Color Palette Generator)
* **Description**: A developer-focused color palette designer.
* **Key Features**: HSL/LCH-based harmonious color generation (monochromatic, analogous, split-complementary), WCAG contrast ratio analysis, color-blindness simulation modes, and CSS/Tailwind variables export.

### 4. Diff Master (Text Diff & Merge)
* **Description**: A side-by-side comparison tool for code and text files.
* **Key Features**: Visual inline or split differences highlighting additions, edits, and deletions, interactive merge arrows, and file upload support for standard code files.

### 5. DevPlayground (Offline Code Sandbox)
* **Description**: A lightweight HTML/CSS/JS sandbox.
* **Key Features**: Multi-pane editor with autocomplete and formatting, isolated iframe output container, and local console log capture.

### 6. Cron Composer (Visual Cron Builder)
* **Description**: A human-readable cron expression generator.
* **Key Features**: Interactive sliders/dropdowns for cron parameters, live natural language explanation (e.g., "At 04:05 on every Tuesday"), and list of next 5 execution times.

### 7. FormatForge (Batch Encoder / Decoder)
* **Description**: A conversion tool for developer data.
* **Key Features**: Multi-format support (Base64, URL encoding, Hex, Binary, Morse code, HTML entities), batch processing input, and character/byte counters.

### 8. Boardroom (Vector Whiteboard)
* **Description**: A canvas-based diagram sketchpad.
* **Key Features**: Vector drawing tools (lines, circles, boxes, text), hand-drawn sketchy style mode, and SVG/PNG export options.

### 9. MarkSlides (Markdown Slide Deck Maker)
* **Description**: Write simple Markdown to generate a presentation slideshow.
* **Key Features**: Real-time slide preview, transition style toggles, presenter notes panel, fullscreen presenter window, and PDF printing layout.

### 10. CSV Studio (Data Grid & Formatter)
* **Description**: Offline tabular data editor and formatter.
* **Key Features**: Excel-like sheet editor for CSVs, column sorting/filtering, cells find/replace, and export formats (cleaned CSV, JSON array, HTML Table).

### 11. CSS Glassmorphism Generator
* **Description**: Visual CSS glassmorphism builder.
* **Key Features**: Interactive sliders for blur, transparency, color, and border-radius, real-time background selection, and copy-pasteable CSS codes.

### 12. Regex Cheat Sheet & Trainer
* **Description**: Learning guide and interactive training tool for regex.
* **Key Features**: Interactive mini-games matching words, step-by-step regex lessons, and searchable cheat sheet for tokens/flags.

### 13. ASCII Art Converter
* **Description**: Convert images or text into retro ASCII/text art.
* **Key Features**: Threshold sliders for character density, font size adjustments, character set options (blocks, letters, standard characters), and copy-to-clipboard.

### 14. JSON Schema Validator
* **Description**: Offline JSON schema inspector.
* **Key Features**: Input JSON and corresponding schema to validate structure and data types, side-by-side editing, and detailed error list highlighting line numbers.

### 15. Programmer Base Converter
* **Description**: Real-time numerical converter.
* **Key Features**: Interlinked input fields for Binary, Octal, Decimal, Hexadecimal, and ASCII, support for signed/unsigned numbers, and bitwise logic indicators.

### 16. Readability & Text Statistics
* **Description**: Text stats and readability analyzer.
* **Key Features**: Detailed metrics (word count, sentence count, reading time, speaking time), Flesch-Kincaid reading ease score, and top keyword density list.

### 17. Dummy Data Generator
* **Description**: Generate customizable placeholder datasets.
* **Key Features**: Generate lists of names, emails, addresses, or phone numbers, customizable schemas, and export support for JSON, CSV, and SQL insertions.

### 18. ShortLink Builder (Offline URL Shortener & Redirect Generator)
* **Description**: Create and manage custom short redirect links locally.
* **Key Features**: Generate tiny portable HTML redirect files, compile list of redirects to Netlify `_redirects` or Vercel `vercel.json` config syntax, create Base64/compressed URL share links, and store mapping databases locally.

### 19. Keyboard Keycode Tester
* **Description**: Interactive input key event inspector.
* **Key Features**: Press any key to see Javascript event outputs (`e.key`, `e.code`, `e.which`), key combinations logger (detects modifier flags), and virtual keyboard highlight.

### 20. JWT Inspector
* **Description**: Decode and inspect JSON Web Tokens locally.
* **Key Features**: Decodes header, payload, and signature JSONs without sending data online, validation indicators, and date-time converters for expiry timestamps.

## Part 2: 10 Puzzle/Shooter Game Ideas

### 1. Grid-Breaker Command
* **Description**: Action puzzle vertical shooter where block cascades descend. Shoot blocks from your ship at the bottom to alter their colors or blast them away to complete rows and clear lines.

### 2. Refractor
* **Description**: Laser bounce puzzle shooter where you aim and fire lasers to clear targets by rotating and shifting reflective glass block nodes.

### 3. Quantum Collapse
* **Description**: Shoot elemental block charges into a dynamic rotating grid. Match 3 or more blocks of the same type to collapse sections before the cluster reaches your emitter.

### 4. Bastion Defense
* **Description**: Build castle structures dropping Tetris-like blocks in a build phase, then defend them from incoming hordes using turrets in a shooting phase.

### 5. Magnetic Pulse
* **Description**: Use an electromagnetic gravity gun to pull, push, and launch magnetic blocks into matching socket grids in physics chambers.

### 6. ChronoShot
* **Description**: Time-control puzzle shooter. Enemy bullets and nodes only move when your ship moves, requiring precise step-by-step navigation and shot planning.

### 7. Word-Slinger
* **Description**: Anagram shooter. Shoot block letters into horizontal rows to spell valid words, which collapses the row and scores bonus points.

### 8. Prism Deflector
* **Description**: Fire primary color laser beams at prism blocks, mixing colors (e.g. Red + Blue = Magenta) by reflecting them through mixers to hit target color sockets.

### 9. Sector Collapse
* **Description**: Solve block grid layouts by shooting unstable power blocks to initiate chain-reaction explosions, clearing the puzzle board in as few shots as possible.

### 10. Cyber-Shield
* **Description**: Breakout-style defender. Protect a central node from incoming block waves using a reflective paddle shield, reflecting blocks back to damage spawn gates.
